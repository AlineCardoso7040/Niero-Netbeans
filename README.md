Visão Geral do Projeto

O Projeto Niero é uma solução de software desenvolvida para digitalizar e otimizar os processos administrativos e operacionais da Niero Assessoria, uma empresa de serviços de consultoria.
O objetivo principal é substituir a gestão manual e pouco integrada por um sistema robusto que garanta eficiência, organização e rastreabilidade dos dados.
A versão atual utiliza MySQL como sistema de gerenciamento de banco de dados, garantindo maior escalabilidade e segurança para o crescimento da empresa.

 Funcionalidades Principais

O sistema é focado na gestão de clientes e no ciclo de vida das consultorias:

-Gestão de Clientes: Cadastro e manutenção de informações detalhadas (nome, CPF/CNPJ, contato, endereço).

-Agendamento de Consultorias: Controle centralizado de data, hora, cultura e status, eliminando conflitos de agendamento.

-Registro de Ações Recomendadas: Campo crucial para registrar o resultado da consultoria, servindo como base para futuros relatórios de desempenho e acompanhamento.

-Interface Intuitiva: Aplicação Desktop desenvolvida em Java para usabilidade e estabilidade local.
